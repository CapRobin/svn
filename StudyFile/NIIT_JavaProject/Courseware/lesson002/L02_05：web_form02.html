web-form表单

表单主体
用途 元素名称 固定属性
圈定表单元素 form action="{服务端处理程序的地址}"
指定提交方式 form method="get/post"
发起提交请求 input type="submit"
本质：将表单元素，以键值对的形式，发送给服务端程序
用来填的表单元素
用途 元素名称 固定属性
文本 input type="text"
数字 input type="number" min="" max=""
邮箱 input type="email"
url input type="url"
密码 input type="password"
大段文字 textarea rows="{可见行数}"
  1. 必须设置name属性
  2. 用value属性获取用户的输入，也可以提前初始化
用来选的表单元素
用途 元素名称 固定属性
单选 input type="radio"
多选 input type="checkbox"
下拉列表 select
组合框 input list="{datalist的id}" （在datalist里面定义数据列表）
日期 input type="date" min="" max=""
时间 input type="time" min="" max=""
  1. 同一组的选项元素，name属性必须相同
  2. input实现的选择元素，用checked来选中
  3. select元素，用selected来选中
辅助元素/属性
用途 元素名称 固定属性
辅助获取焦点 label for="{对应表单元素的id}"
关闭自动完成 form autocomplete="off"
默认的描述 placeholder="{默认描述文字}"
自动获取焦点 autofocus="autofocus"
必填验证 required="required"
限制输入字符 maxlength="{允许输入的字符长度}"
提供更好的用户体验
